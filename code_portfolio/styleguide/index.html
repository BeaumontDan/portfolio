Text Styles

H1: Main Page Heading

  • Font‐family: Anton
  • Font‐weight: 400(regular)
  • Font‐size: vw Resonsive
  • Use: For main page and site location headings only

H2: Secondary Headings

  • Font‐family: Antonio
  • Font‐weight: 400(regular)
  • Font‐size: 2em
  • Use: Secondary page headings and portfolio project titles

H3: Tertiary Headings

  • Font‐family: Antonio
  • Font‐weight: 200(lighter)
  • Font‐size: 1.4em
  • Use: Paragraph and Article Titles

H4: Portfolio Grid Headings

  • Font‐family: Antonio
  • Font‐weight: 400(regular)
  • Font‐size: 1.5em
  • Use: Portfolio Thumbnail Titles

p.firstPara: Page Intro

  • Font‐family: Nunito‐Sans
  • Font‐weight: 700(bold)
  • Font‐size: 1.3em
  • Use: Section and page intro paragraphs

Paragraphs

  • Font‐family: Nunito‐Sans

  • Font‐size: 1.1em
  • Use: Body copy and paragraphs

Small Paragraphs
