INTRODUCTION TO HTML

Welcome to < CHEATSHEET >, my HTML quick‐ref‐guide to the basics of HTML & CSS. I hope you find it useful.

Lets start with the basics of HTML and its essential tags and structure. This should provide you with enough to write your first web page.

HTML ESSENTIALS

ESSENTIAL TAGS

TAG NAME DESCRIPTION
<DOCTYPE! html> Doc Type Defines the page as HTML mark up and must be the first tag in all HTML documents.
<html> HTML Defines the doument as a web page. All other dags must fall within the HTML tags
<head> Head Contains none‐display conent and meta‐data, including page title, linked stylesheets and javascript functions.
<title> Title The title tag defines the page title that will appear in at the top of the browser window or in page tabs.
<body> Body Typically contains all content to be displayed to end users.

ESSENTIAL TAGS SAMPLE

<DOCTYPE! html>

<html>

  <head>
    <title>Page Title</title>
  </head>
                          
  <body>
            
    This is where all web page content is placed.

  </body>
                          
</html>

HYPERLINKS

ANCHOR TAGS

TAG NAME DESCRIPTION
<a> Anchor Tag Defines a clickable link. Anchor tags can be placed around almost all other tags, converting any element of group of elements into a link

LINK ATTRIBUTES

ATTRIBUTE NAME DESCRIPTION
<a href="..."> HRef (or Hyperlink Reference) This attribute is required for all anchor tages.
The value dictates the location, which when clicked, the user will be directed to. The vaule can be:
An external page ‐ "https://www.google.com/"
A relative/internal page ‐ "./index.html/"
A location on a page link using the element id "#footer"
<a name="..."> Link Name Name of the link (name of anchor or name of bookmark)
<a target="..."> Target Location Defines where a clicked link will open, values include:"_self", "_blank", "_top" and "_parent"
<a href="url#bookmark"> Bookmark Link to a bookmark defined by the name attribute
<a href="mailto:email@email.com"> Email link Initiate, when clicked, an email through the users email client
<a href="tel:+441234567890"> Telephone link Initiate a phone call, depending on users device

LISTS

LIST TAGS

TAG NAME DESCRIPTION
<ul> Unordered List Defines an unordered list, bullet‐pointed by default
<ol> Ordered List Defines an ordered list, numbered by default
<li> List Item Defines a list item (within an ordered or unordered list)
<dl> Definition List Defines a definition item
<dt> Term A Term of phrase to be defined
<dd> Detailed Definition Defines a detailed definition list item

LIST ATTRIBUTES

ATTRIBUTE NAME DESCRIPTION
<ol type=" "> Ordered List Type Sets the list type with values A, a, I, i or 1
<ol start=" "> Ordered List Starting Value Allows you do dictate the starting value for the list
<ul type=" "> Unordered List Type Sets the unordered list type with values disc, circle or square
<li value=" "> List Item Value Changes current and subsequent items
<li type=" "> List Item Type Changes ONLY the current item

IMAGES

IMAGE TAG

TAG NAME DESCRIPTION
<img> Image Defines the placement of an image, through the src attribute. The img tag is relitivley unique in that it is self closing and doesn't require a </img> tag

IMAGE ATTRIBUTES

ATTRIBUTE NAME DESCRIPTION
<img src="..."> URL or File Path and Name This attribute is required for all Image tages.
<img alt="..."> Alternative Name This is required for all image tags and provides altnerative text for text only browsers
<img align="..."> Align Aligns image within surronding text (*)
<img width="..."> Width Defines the image width in pixels or % (*)
<img height="..."> Height Defines the image height in pixels or % (*)
<img border="..."> Border Defines border thicknes in pixels (*)
<img vspace="..."> Vertical Space Defines the space above and below the image in pixels (*)
<img hspace="..."> Horizonal Space Defines the space left and fight of the image in pixels (*)

SEMANTIC MARKUP

SEMANTIC TAGS

TAG NAME DESCRIPTION
<head> Head Contains none‐display conent and meta‐data, including page title, linked stylesheets and javascript functions.
<body> Body Typically contains all content to be displayed to end users.
<header> Page Header Defines the header section of the webpage, commonly contains page title or logo and/or navigation links
<section> Section Defines a section within a page and should contain a heading title
<article> Article Defines an article within a section and should contain a heading title. Helps to group related elements.
<aside> Aside Defines a side note to surrounding elements/content
<figure> Figure Container for stand‐alone images and videos
<blockquote> Long Quote Defines a long quote.
<footer> Footer Defines the page footer, the final element to a page and commonly contains copyright info, contact details and links

SEMANTIC MARKUP SAMPLE

<html lang="en">

  <head>
    <title>Page Title</title>
  </head>
                          
  <body>
                          
    <header>
      Logo, title, navigation

      <nav>
        Primary and secondary menus but ever nest navs
      </nav>
              
    </header>
               
    <section>
      Related grouping of semantic meaning
                          
      <article>
        Self-contained content
      </article>
                          
      <article>
        Would still make sense if all other content removed

        <aside>

          <blockquote>
            Related to main but outside the main flow
          </blockquote>

        </aside> 

      </article>
      
    </section>
                          
    <footer>
      Copyright, contact, links
    </footer>
                          
  </body>
                          
</html>

TABLES

TABLE TAGS

TAG NAME DESCRIPTION
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<th> Table Heading Sets the title for each column of data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The set of rows defining the footer in a table.

TABLE ATTRIBUTES

ATTRIBUTE NAME DESCRIPTION
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span The set of rows defining the column headers in a table.

TABLE SAMPLE

FORMS

FORM TAGS

TAG NAME DESCRIPTION
<form> Form The wrapper element for all HTML forms.
<fieldset> Field Set Descripes the fieldset
<input> Input Indicates a user input requirement, defined by <type> attribute
<label> Label Provides users with a label for the input required.

FORM SAMPLE

<form>

  <fieldset>
    <legend>Describes fieldset</legend>
                      
    <p>
      <input type="radio" name="vertical" id="option1" value="top">
      <label for="option1">Top</label>
    </p>
                      
    <p>
      <label for="option2">
      <input type="radio" name="vertical" id="option2" value="bottom"> Bottom
      </label>
    </p>
                      
    <p>
      <label for="horizontal">
        Horizontal:
        </label>
        <select id="horizontal" name="horizontal">
          <option value="left">Left</option>
          <option value="middle">Middle</option>
          <option value="right">Right</option>
        </select>
    </p>
                      
  </fieldset>
                      
 </form>
