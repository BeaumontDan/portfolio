STYLE TYPES

There are three options available to style your HTML cosuments:
Inline Style Inline styles are placed directly into the HTML tag being styled.
They override all other styles but are specific to the individual element and as such are not reusable.
Internal Stylesheet Placed within the <head> tags of your document, they orverrid external stylesheets
but are only attributed to the current HTML document.
External Stylesheets Widley accepted as the best practice, extneral stylesheets allow styling rules to be used throughout an
entire website and allow updates and changes to be made once and carried across all pages.
You can utilies any one or a commbination of all 3 depending on your requirements.

EXTERNAL STYLESHEET

External stylesheets are independent text documents saved with the .css file extension. These are structured/formatted in the same way as the internal stylesheet. These are then linked to your HTML documents using the <link> tag, placed within the <head> element.

            
<head>

  <link rel="stylesheet" type="text/css" href="style.css" title="style">

</head>


* Please note: Whilst these tags/attributes will still work, it is now best practice to set these attributes via CSS.
