USING STYLES

STYLE TYPES

There are three options available to style your HTML cosuments:
Inline Style Inline styles are placed directly into the HTML tag being styled.
They override all other styles but are specific to the individual element and as such are not reusable.
Internal Stylesheet Placed within the <head> tags of your document, they orverrid external stylesheets
but are only attributed to the current HTML document.
External Stylesheets Widley accepted as the best practice, extneral stylesheets allow styling rules to be used throughout an
entire website and allow updates and changes to be made once and carried across all pages.
You can utilies any one or a commbination of all 3 depending on your requirements.

INLINE STYLE

Inline styles are placed into the element tag using the style=" " attribute.
Each CSS attribute/value must be punctuated with the ; semi colon.

            
<p style="font‐family: Helvetica; color: red;">Your paragraph text here</p>

INTERNAL STYLESHEET

Internal Stylesheets are placed within the <head> tags using the <style type="text/css"> tag and attribute.
Each CSS selector must be followed by opening and clossing curly brackets {}, with attribute: value; placed within.

            
<head>

  <style type="text/css">

    h1 {
      font‐family: Helvetica;
      color: red;
    }

  </style>

</head>

EXTERNAL STYLESHEET

External stylesheets are independent text documents saved with the .css file extension. These are structured/formatted in the same way as the internal stylesheet. These are then linked to your HTML documents using the <link> tag, placed within the <head> element.

            
<head>

  <link rel="stylesheet" type="text/css" href="style.css" title="style">

</head>

COLORS & BORDERS

SETTING COLORS

ATTRIBUTE : VALUE; DESCRIPTION
color: red; Element (usually text) color.
background‐color: blue; Background color of an element or container
background‐image: url(image.jpg); Sets the point size of your text, 12pt, 20px, 2em
border‐color: green; Sets the border color for an element
border: 1px solid yellow; Border shorthand ‐ sets width, style and color.

COLOR TYPES

There are several options available when setting CSS colors:
color: red; Name / Keyword There are over 150 named colors, including red, blue, yellow, green, black and white. You can find a full list here.
color: #ffffff Hex (Hexadecimal) Hex colors are specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) integers specify the components.
color: rgb(255, 0, 0); RGB Each parameter (red, green, and blue) defines the intensity of the color between 0 and 255
color: rgba(255, 0, 0, 0.5); RGBA RGBA color values are an extension of RGB color values with an alpha channel ‐ which specifies the opacity for a color.
The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).
color: hsl(0, 100%, 50%); HSL Hue, saturation, and lightness:
Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.
Saturation is a percentage value. 0% means a shade of gray, and 100% is the full color.
Lightness is also a percentage. 0% is black, 50% is neither light or dark, 100% is white
color: hsl(0, 100%, 50%, 0.5); HSLA Like with RGBA, the A value in HSLA is the Alpha channel which specifies the opacity for a color.
The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).
